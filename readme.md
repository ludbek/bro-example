An example project template.

## Requirements
### 1. [bro](https://github.com/ludbek/bro)
### 2. tmux
### 2.1. Mac
`brew install tmux`
### 2.2. Ubuntu
`sudo apt-get install tmux`

## Usage
### 1. To setup new project based upon this template
`bro create -t https://github.com/ludbek/bro-example.git aproject`

### 2. To quit
`bro exit`

### 3. To workon
`bro start aproject`
